UE5 루멘 활용하기: 기존 UE4 프로젝트에서 넘어왔을 때 주의사항

  • November 10, 2021
  • Posted By 김경환
  • Unreal engine , Unreal engine 5 , Lumen

UE5 루멘에 대해 저와 함께 공부하고 싶으신 분들은 우측 Categories에서 Lumen을 클릭해주세요!

UE4 프로젝트를 UE5에서 열었을 때 주의사항

언리얼 엔진 4 프로젝트를 언리얼 엔진 5 얼리 액세스로 업그레이드 하는 방법은 마이그레이션 가이드 | 언리얼 엔진 5 얼리 액세스 문서를 참고하세요!

언리얼 엔진 4로 작업하다가 언리얼 엔진 5로 넘어오신 저와 같은 분들이 꼭 확인하실 사항이 있습니다. 언리얼 엔진 4로 생성한 프로젝트를 언리얼 엔진 5에서 실행시켰을 땐 루멘이 비활성화되어 있습니다!!! (언리얼 엔진 5에서 생성한 프로젝트는 루멘이 디폴트로 활성화 되어 있으니 걱정 안 하셔도 됩니다.)

따라서 위와 같은 분들은 프로젝트 세팅에서 Dynamic Global Illumination Method(다이내믹 글로벌 일루미네이션 메서드)를 Lumen(루멘)으로 설정하셔야 비로소 루멘이 적용된 씬을 보실 수 있습니다.

  1. Project Settings

  2. Engine -> Rendering -> Global Illumination -> Dynamic Global Illumination 이미지가 잘 안 보일 경우 우클릭 후 ‘새 탭에서 이미지 열기'로 봐주세요!

    Dyanamic Global Illumination을 Lumen으로 설정해주세요.
    자동적으로 Reflections 메뉴 아래 Reflection Mode도 Lumen으로 설정됩니다.
    자동적으로 Software Ray Traicing 메뉴 아래 Generate Mesh Distance Fields가 활성화됩니다.

  3. 엔진을 다시 시작하면 Mesh마다 Distance Field가 생성이 되고 루멘의 Software Ray Tracing 방식 라이팅을 사용하실 수 있습니다.
