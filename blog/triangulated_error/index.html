How to troubleshoot 3ds Max export triangulating your mesh

When exporting your mesh as fbx in 3ds Max, the mesh becomes triangulated. You can check them in whatever tools you import the mesh, including 3ds Max itself.



There are several ways to troubleshoot this problem, but I use Turn to Poly modifier.



Add Turn to Poly modifier in the stack and enable Limit Polygon Size. Set the Max Size to 4, which will make the polygons to stay in quads. It’s the max size, therefore, no triangles you created on purpose will be affected.
